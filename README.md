# Dobie 
> 초보자를 위한 인프라 서비스 플랫폼

Dobie는 초보자도 쉽게 사용할 수 있는 인프라 서비스 플랫폼입니다.

사용자는 간단한 정보만을 입력하고 자신의 프로젝트를 배포하는 경험을 할 수 있습니다.

Dobie는 Docker, Docker Compose를 활용하여 사용자의 프로젝트를 컨테이너로 관리합니다.
